# Документация модуля 6 - Архив отчетов

## Описание
Модуль 6 предназначен для просмотра и управления архивом сохраненных отчетов, которые автоматически генерируются модулем 3.

## Функциональность

### Основные возможности:
1. **Просмотр списка отчетов** - отображает все сохраненные отчеты, отсортированные по дате (новые сверху)
2. **Просмотр содержимого отчета** - позволяет открыть и просмотреть полное содержимое выбранного отчета
3. **Удаление отчетов** - возможность удалить ненужные отчеты из архива
4. **Автоматическое обновление** - кнопка обновления для подгрузки новых отчетов

### Интерфейс:
- **Левая панель**: Список всех доступных отчетов с датой и временем создания
- **Правая панель**: Область просмотра содержимого выбранного отчета
- **Верхняя панель**: Заголовок и кнопка "Обновить"
- **Нижняя панель**: Информация о выбранном отчете и кнопка "Удалить отчет"

## Взаимодействие с модулем 3

### Автоматическое сохранение отчетов:
- Модуль 3 автоматически сохраняет отчеты каждую минуту
- Отчеты сохраняются в папке `reports/` в формате txt
- Имя файла: `report_YYYY-MM-DD_HH-MM-SS.txt`

### Содержимое отчетов:
Каждый отчет содержит:
- Дату и время создания отчета
- Время измерения данных
- Параметры нефтепровода (давление, температура, скорость откачки, вибрации, угол наклона)
- Параметры окружающей среды (температура, давление, скорость ветра, влажность)

## Технические особенности

### Файловая структура:
```
reports/
├── report_2025-06-18_13-18-28.txt
├── report_2025-06-18_13-17-28.txt
└── ...
```

### Формат отчета:
```
Дата создания отчета: 2025-06-18 13:18:28
==================================================

ОТЧЕТ О СОСТОЯНИИ НЕФТЕПРОВОДА
========================================

Время измерения: 2025-06-18 12:05:53

ПАРАМЕТРЫ НЕФТЕПРОВОДА:
------------------------------
Давление: 50.65 МПа
Температура: 17.30 °C
Скорость откачки: 5.24 м/с
Вибрации: 0.06 мм/с
Угол наклона: 8.25 градусов

ПАРАМЕТРЫ ОКРУЖАЮЩЕЙ СРЕДЫ:
------------------------------
Температура: 15.00 °C
Давление: 101.30 кПа
Скорость ветра: 3.00 м/с
Влажность воздуха: 60.00 %
```

## Использование

1. **Просмотр отчетов**: Выберите отчет из списка слева, его содержимое отобразится справа
2. **Обновление списка**: Нажмите кнопку "Обновить" для подгрузки новых отчетов
3. **Удаление отчета**: Выберите отчет и нажмите "Удалить отчет", подтвердите действие

## Интеграция с основным приложением

Модуль 6 интегрирован в основное приложение MonOilStudy и:
- Автоматически инициализируется при разблокировке модулей
- Работает независимо от подключения к базе данных
- Использует стандартную архитектуру модулей приложения

## Настройки модуля 3 для работы с архивом

### Периодичность генерации отчетов:
- **Изменено с**: 200 мс (5 раз в секунду)
- **Изменено на**: 60000 мс (1 раз в минуту)

### Автоматическое сохранение:
- Отчеты сохраняются автоматически каждую минуту
- Данные берутся из подключенной базы данных
- Используется кодировка UTF-8 для корректного отображения русского текста
